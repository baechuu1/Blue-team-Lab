🛡️ #Blue Team Detection Lab
📌 #Overview

This project is a hands-on blue team lab designed to simulate real-world cyber attacks and demonstrate detection and analysis using Splunk Enterprise.

Each scenario includes:

Attack execution
Log analysis
Detection queries
MITRE ATT&CK mapping
Mitigation strategies

🧪 Lab Environment

Kali Linux (Attacker)
Ubuntu Server (Target)
Splunk SIEM (Log Analysis)
Host-only network (internal attack simulation)

⚔️ Attack Scenarios

Attack	Description
SSH Brute Force	Credential attack via Hydra
SQL Injection	Web exploitation
Password Spraying	Low-and-slow auth attack
Lateral Movement	Internal pivoting
Persistence	Maintaining access
Beaconing	Command & control simulation

🎯 Objective

To simulate attack scenarios and develop detection strategies using log analysis and SIEM tools.
